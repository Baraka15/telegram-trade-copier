# Telegram Trade Copier

Developed by **Baraka Emmanuel**, Telegram Automation Engineer based in Kampala, Uganda.

## Overview

This project is a real-time Telegram to MetaTrader 5 trade copier designed for deterministic, low-latency signal execution.

It connects Telegram channels to MT5 and executes trades automatically using structured signal parsing and risk control logic.

---

## Core Features

- Real-time Telegram listener (Telethon)
- Intelligent signal extraction engine
- Deterministic MT5 execution gateway
- Risk-adjusted position sizing
- Trade logging and failure handling

---

## System Architecture

Telegram Channel  
→ Signal Parser  
→ Risk Engine  
→ MT5 Execution Layer  
→ Broker

---

## Technology Stack

- Python
- MetaTrader5 API
- Telethon
- NumPy
- AsyncIO
